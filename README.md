# AcadEvents

## Sobre o Projeto

O AcadEvents é uma plataforma desenvolvida para facilitar a gestão de eventos acadêmicos, centralizando inscrições, controle de presença, emissão de certificados e relatórios em um único sistema.

## Problema

Atualmente, muitas instituições de ensino utilizam diferentes ferramentas para gerenciar eventos acadêmicos, como formulários online, planilhas, e-mails e listas de presença impressas. Isso gera retrabalho, dificuldades de organização e maior risco de erros.

## Solução

O AcadEvents centraliza todo o processo em uma única plataforma, permitindo:

* Cadastro de eventos
* Inscrições online
* Check-in por QR Code
* Controle de presença
* Emissão automática de certificados
* Relatórios em tempo real
